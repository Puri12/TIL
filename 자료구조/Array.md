# Array

## 정의

> 배열은 연속된  메모리 공간에 순처적으로 저장된 데이터 구조

## 특징

- **동일한 데이터 유형**을 가지고 있음
- **연속적인 메모리 공간**에 **순차적**으로 저장됨
- index를 통한 점근이 용이함
- 배열은 선언할 때 크기를 정하면 이후에 변경할 수 없음
- 배열의 주소를 보면 한 칸마다 배열의 자료형의 크기를 가지고 있음

## 사용환경
`정적인 데이터를 활용하면서 조회가 빈번할 때 사용하기 좋음`

## 배열의 시간 복잡도

| Operation | average case | worst case
| --- | --- | --- |
| read | O(1) | O(1)
| insert | O(n) | O(n)
| delete | O(n) | O(n)
| search | O(n) | O(n)
